# Weather App (Next.js)

Minimal Next.js app that fetches current weather from OpenWeatherMap.

Quick start

1. Change into the app folder:

```bash
cd weather-app
```

2. Install dependencies:

```bash
npm install
```

3. Copy `.env.example` to `.env.local` and add your OpenWeatherMap API key:

```bash
cp .env.example .env.local
# edit .env.local and set NEXT_PUBLIC_OPENWEATHER_API_KEY
```

4. Run in development:

```bash
npm run dev
```

Open http://localhost:3000 to view the app.
